**SubDecode** is a simple command-line tool to fetch, decode, deduplicate, and manage subscription links (VLESS/VMESS/Trojan/Shadowsocks). It automatically saves output to a file, updates it if content changes, and copies it to the clipboard.

  - Unchanged nodes
  - Removed nodes
